| T-12-02-02 | Information Disclosure | server most-skipped list | mitigate | get_leaderboard_skips already keyed on `guild_id = $1`; no cross-guild rows |
| T-12-02-03 | Tampering | song_history aggregate SQL | mitigate | All values bound as $1/$2 positional params; no string interpolation |
| T-12-02-04 | Integrity (misleading stat) | tiny-sample 100% rates | mitigate | SKIP_STATS_MIN_PLAYS floor in pure logic suppresses rates below the threshold (D-08) |
| T-12-02-05 | DoS | slow aggregate on a large guild | mitigate | Inherits the DB command timeout; /skips catches asyncio.TimeoutError and degrades gracefully (mirrors /leaderboard) |
| T-12-02-SC | Tampering | package installs | n/a | No new packages (stdlib + already-installed deps only) |
