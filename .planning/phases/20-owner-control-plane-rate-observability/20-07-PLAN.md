| Boundary | Description |
|----------|-------------|
| any user → /guilds subcommand | Commands are globally visible (D-06); a non-owner may invoke — the inline is_owner gate is the real access control |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-20-SC | Tampering (supply chain) | package installs | mitigate | Zero new installs; nothing to audit |
